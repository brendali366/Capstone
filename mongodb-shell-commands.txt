common commands to use in mongodb terminal: https://www.youtube.com/watch?v=c2M-rlkkT5o&list=WL&index=1 

cls - clear screen

use database_name - switch to a database_name or create a new database

db.createCollection("collection_name") - create a collection
db.dropDatabase() - to drop a database. you'll need to be in it first.
db.collection_name.insertOne({name:"name1",age:"age1,fieldname:"exact_field_name"}) - insert as many field value pairs you want in the collection
- use .insertMany([{},{},{}]) - insert many documents. (more than one).
db.collection_name.find() - will show you all documents in the collection name
db.getCollectionNames() - gives all the collections in the database.

Naming conventions for collections:
-PrimaryUsers = kids
-ContactBook = stores secondary user information

video timestamps:
20.11 - nested documents 

sorting and limiting:
db.collection_name.find().sort({field_name:1}) - sort names in alphabetical order, -1 is in descending order
^can be use ful for sorting leaderboard scores.
db.collection_name.find().limit(#) - can limit how many documents come up
db.collection_name.find().sort({}).limit() - to use both.

find method (.find()):
db.collection_name.find({query}, {projection}) ex: db.PrimaryUsers.find({}, {_id:false, name:true}) - will return all documents but only with name
db.PrimaryUsers.find({}, {_id:false, name:true, score:true}) - will return name, with leaderboard score
^useful for leaderboard feature.

update documents:
db.collection_name.updateOne({filter},{$set:{update}}) i.e db.collection_name.updateOne({name:"brenda"},{$set:{email:"hi@gmail.com"}})
-.updateMany() is also an option
- use user id instead of name (in case of people with the same name)
